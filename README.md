# 🎬 Netflix Clone Project

This project is a responsive web application inspired by the look and feel of Netflix. 
It was created as a study project focusing on modern HTML, CSS, and JavaScript/TypeScript.

## 🚀 Live Demo
[👉 Click here to view the live demo of the application](https://jirigalacz.github.io/Engeto-Projekt-Netflix/)

## ✨ Main Features
* **Responsive Design**: The website looks great on mobile devices, tablets, and large desktop monitors (utilizing CSS Flexbox and Grid).
* **Movie Gallery (API)**: Real-time searching and dynamic loading of TV shows/movies using the [TVMaze API](https://www.tvmaze.com/api).
* **Advanced Form Validation**: 
  * Email and phone number format checking.
  * Dynamic password matching validation with real-time visual feedback.
* **Modern UI Elements**: Sticky header, smooth scroll-to-top button, and interactive hover effects.

## 🛠️ Technologies Used
* **HTML5 & CSS3**: Semantic structure and modern styling (CSS variables, custom properties).
* **TypeScript / JavaScript**: Application logic compiled into ES modules.
* **REST API**: Asynchronous data fetching (`fetch`, `async/await`).
